git中生成和使用patch

2018-10-24

一个比较奇特的情景：有两个本地仓库A和B，现在需要将其中一个仓库A的几个提交应用到另一个仓库B上。
有一种思路是使用git的cherry-pick：

  1. 首先在B的远程仓库地址中添加A；
  2. 将A中对应的分支fetch到B中；
  3. 在B里使用cherry-pick应用A的提交。

但我在这里要记录另一种方法。
